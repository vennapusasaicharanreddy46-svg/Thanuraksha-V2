# 🏥 THANURAKSHA – AI-Powered Smart Healthcare Platform

THANURAKSHA is a comprehensive healthcare platform designed to connect patients, doctors, pharmacies, and administrators through a secure and intelligent mobile application. The project integrates AI-powered health assessment tools, appointment management, medical records, and Firebase cloud services to provide an efficient digital healthcare ecosystem.

# 📦 Installation

Clone the repository

```bash
git clone https://github.com/your-username/thanuraksha.git
```

Move into the project

```bash
cd thanuraksha
```

Install dependencies

```bash
npm install
```

Start the project

```bash
npx expo start
```

For Android

```bash
npx expo run:android
```

---

# 🔥 Firebase Setup

1. Create a Firebase Project
2. Enable Authentication
3. Enable Realtime Database
4. Enable Storage
5. Download

```
google-services.json
```

6. Place it inside

```
android/app/
```

7. Update Firebase configuration inside

```
config/
```
